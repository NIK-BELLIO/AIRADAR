Step 2 — Choose connection method

No webhook, no token, no developer dashboard for users. Connect Instagram or do not use publishing.

Open Meta Apps Open Graph API Explorer Open Token Debugger
If Meta says “not available in this location” This is a Meta account/location restriction, not an AI Radar error. Changing VPN alone may not work because Meta can also use your account country, phone region, cookies, browser session, payment/business profile, device signals, and recent login history.
1. Try a clean browser profile or incognito window, then open Meta Developers again. 2. Keep one stable allowed country/IP. Do not switch between many VPN countries quickly. 3. Log in with the Facebook account that owns the Page connected to the Instagram Business/Creator account. 4. If the same error remains, create/get the token from a trusted device or teammate located in an allowed region, then paste only the token and IG User ID here. 5. You can still use AI Radar as a draft calendar until Meta Developer access is available.

Step 3 — Connect publishing

For normal users: click Connect Instagram directly. The webhook field is only for advanced admin/provider setups.

Normal users click this button, approve Instagram, and return to AI Radar. No webhook, no token copy, no Meta Developer page.

Owner setup: activate this button for all users

Add the same Redirect URI in your Instagram app settings. After saving, every user can connect with the single button above.

Advanced: webhook/provider setup

Recommended: AI Radar sends caption, mediaUrl, mediaType, and scheduledAt to this endpoint when a post is due. Use Make, Zapier, Pipedream, Ayrshare, or another approved Instagram publishing provider.

Advanced: manual Meta token

Official Meta token fallback. Never paste your Instagram password.
