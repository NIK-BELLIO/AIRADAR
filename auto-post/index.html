Meta Apps Graph API Explorer Token Debugger Content Publishing Docs
1. Prepare the Instagram account

Switch Instagram to Business or Creator, then connect it to a Facebook Page.

2. Create or select a Meta app

Open Meta Developers, create/select an app, and add Instagram Graph API access.

3. Generate an access token

In Graph API Explorer, select your app and request these permissions:

instagram_basic instagram_content_publish pages_show_list pages_read_engagement
4. Find the IG User ID

Run these Graph API calls in Explorer. First get your Pages:

GET /me/accounts

Then use the Page ID to get the connected Instagram account:

GET /{page-id}?fields=instagram_business_account

Copy instagram_business_account.id into IG User ID. Copy your generated Meta access token into Meta Access Token. For production, use a long-lived token and confirm it in Token Debugger.
